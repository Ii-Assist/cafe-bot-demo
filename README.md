# Telegram-бот для кафе / ресторана
Готовый Telegram-бот для заведений общепита. Клиент видит меню с фотографиями, бронирует столик, читает акции и оставляет отзывы.
**Демо:** [@helpergou_bot](https://t.me/helpergou_bot)
## Возможности
- Меню по категориям с фото и ценами (19 позиций)
- Бронирование столика (имя, дата, время, гости)
- Акции и спецпредложения
- Информация о заведении
- Отзывы с уведомлением администратору
## Стек
Python 3.10+, python-telegram-bot v20+, ConversationHandler
## Быстрый старт
1. `pip install -r requirements.txt`
2. Вставить токен в config.py
3. `python bot.py`
